# Benjamin Ory

Source code for [benjaminory.com](https://benjaminory.com), Benjamin Ory’s personal academic website.

The site is built with Jekyll and includes research descriptions, publications, news, downloadable PDFs, and resource pages for archival/catalogue materials.

## Contents

- Homepage with biography, research projects, publications, news, and CV link
- Catalogue of correspondence for Armen Carapetyan
- Archival materials page
- Downloadable PDFs and CSV data
- Metadata-driven resource pages generated from JSON/CSV files

## Project Structure

```text
_pages/             Site pages
_includes/          Shared includes, styles, scripts, and metadata
_data/              Jekyll data files
assets/img/         Images and publication previews
assets/pdf/         Downloadable PDFs
assets/csv/         Downloadable CSV data
_layouts/           Jekyll layouts
_config.yml         Main Jekyll configuration
